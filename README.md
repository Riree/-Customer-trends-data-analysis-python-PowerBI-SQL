# Customer Trends Data Analysis

Tech Stack: SQL | Python (Pandas, NumPy, Matplotlib) | Power BI

# Project Overview

This end-to-end data analytics project focuses on understanding customer shopping trends in the retail domain. The objective is to analyze sales performance, product demand, and customer behavior to help businesses make data-driven decisions that improve revenue and retention.

# Project Objectives

Identify top-performing products and sales trends

Analyze customer demographics and purchase behavior

Detect seasonal patterns and peak shopping periods

Create interactive dashboards for business insights

Provide actionable recommendations to optimize sales strategies
